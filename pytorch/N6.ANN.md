# Artificial Neural Networks (ANN) in PyTorch — Detailed Guide

An Artificial Neural Network (ANN) is a computational model inspired by biological neurons. In PyTorch, ANNs are built using modules from `torch.nn`.

---

# 1. What is a Neural Network?

A neural network consists of layers of neurons:

```text
Input Layer → Hidden Layer(s) → Output Layer
```

Example:

```text
       x1
         \
          \
       x2 ---> [Hidden] ---> [Output]
          /
         /
       x3
```

Each neuron performs:

1. Weighted sum
2. Add bias
3. Apply activation function

Mathematically:

z = w_1x_1 + w_2x_2 + \cdots + w_nx_n + b

Then:

a = f(z)

where:

* (x) = input
* (w) = weights
* (b) = bias
* (f) = activation function
* (a) = output of neuron

# 6. Building an ANN

PyTorch networks inherit from:

```python
nn.Module
```

Example:

```python
class ANN(nn.Module):

    def __init__(self):
        super().__init__()

        self.fc1 = nn.Linear(2, 4)
        self.fc2 = nn.Linear(4, 4)
        self.fc3 = nn.Linear(4, 1)

    def forward(self, x):

        x = torch.relu(self.fc1(x))
        x = torch.relu(self.fc2(x))
        x = torch.sigmoid(self.fc3(x))

        return x
```

# 8. Activation Functions

Without activation functions:

```text
Network = Linear Transformation
```

No matter how many layers you add.

---

## ReLU

Most common.

f(x)=\max(0,x)

Example:

```python
torch.relu(x)
```

Input:

```text
[-2, 3, -1]
```

Output:

```text
[0, 3, 0]
```

Advantages:

* Fast
* Reduces vanishing gradients

---

## Sigmoid

Used in binary classification.

\sigma(x)=\frac{1}{1+e^{-x}}

Output range:

```text
0 to 1
```

Example:

```python
torch.sigmoid(x)
```

---

## Tanh

\tanh(x)=\frac{e^x-e^{-x}}{e^x+e^{-x}}

Range:

```text
-1 to 1
```

---

## Softmax

Used for multi-class classification.

Example output:

```text
[2.0, 1.0, 0.5]
```

After softmax:

```text
[0.62, 0.23, 0.15]
```

Probabilities sum to 1.

# 12. Training Loop

The heart of PyTorch.

```python
epochs = 100

for epoch in range(epochs):

    outputs = model(X)

    loss = criterion(outputs, y)

    optimizer.zero_grad()

    loss.backward()

    optimizer.step()

    print(loss.item())
```

---

# 13. What Happens During Training?

### Step 1

Forward pass:

```python
outputs = model(X)
```

Prediction is generated.

---

### Step 2

Loss computation:

```python
loss = criterion(outputs,y)
```

Measures error.

---

### Step 3

Clear old gradients:

```python
optimizer.zero_grad()
```

PyTorch accumulates gradients.

---

### Step 4

Backpropagation:

```python
loss.backward()
```

Computes gradients:

\frac{\partial L}{\partial w}

for every weight.

---

### Step 5

Update weights:

```python
optimizer.step()
```

Gradient descent update:

w_{new}=w_{old}-\eta\frac{\partial L}{\partial w}

where:

* ( \eta ) = learning rate

---

# ANN Workflow in PyTorch

```text
Dataset
   ↓
Tensor Conversion
   ↓
DataLoader
   ↓
Neural Network (nn.Module)
   ↓
Forward Pass
   ↓
Loss Function
   ↓
Backward Pass
   ↓
Optimizer
   ↓
Weight Update
   ↓
Repeat for Epochs
   ↓
Prediction/Evaluation
```

This training cycle—**forward propagation → loss computation → backpropagation → optimizer update**—is the core process behind almost every neural network you build in PyTorch, from simple ANNs to CNNs, RNNs, Transformers, and large language models.
